NVIDIA micro-services

NVIDIA NIM (NVIDIA Inference Microservices) is a set of easy-to-use microservices designed to accelerate the deployment of generative AI models across any NVIDIA-accelerated infrastructure. By visiting build.nvidia.com, developers can access a massive catalog of enterprise-grade AI models — including Llama 3, Mistral, and NVIDIA’s own Nemotron — running on high-performance NVIDIA GPUs.

The Free Tier: how it works

NVIDIA offers a “try-before-you-buy” model designed for prototyping and research. Here is how the free access is structured:

Initial Credits: Upon signing up for the NVIDIA Developer Program, you are typically granted 1,000 free API credits. These credits allow you to make requests to the hosted NIM endpoints without needing your own GPU hardware.
Requesting More: If you exhaust your initial 1,000 credits, you can often request additional credits (bringing the total up to 5,000) by providing a business email. This upgrade also typically includes a 90-day NVIDIA AI Enterprise license, which allows for deeper evaluation of the platform.
Transition to Rate Limits: Recent reports from mid-2025 indicate that NVIDIA has been shifting some users from a strict credit-based system to a rate-limited model for ongoing development. This allows for continuous prototyping (roughly 40 requests per minute) as long as the usage is strictly for non-production, developmental purposes.
Key features of the Free catalog
OpenAI-Compatible API: All models in the catalog follow the standard OpenAI API format. You can integrate them into existing applications by simply changing the base_url to NVIDIA's endpoint and providing your API key.
Model Diversity: You aren’t limited to just text LLMs. The catalog includes specialized models for:
Reasoning and Chat: Llama 3.3, Mistral, and Nemotron.
Vision and Multimodal: Models capable of processing images and video.
Coding: Specialized models like DeepSeek-Coder and Qwen-Coder.
BioNeMo: Specialized models for drug discovery and protein folding.

3. Portability: A unique feature of NIM is that once you’ve prototyped using the free cloud API, you can download the same NIM container to run on your own local NVIDIA hardware (RTX PCs or workstations) or private cloud.

Of course… there are some usage restrictions

While the API is free to use for development, NVIDIA distinguishes between “Development” and “Production”:

Free/Developer Tier: Ideal for students, researchers, and hobbyists building proof-of-concepts. You get access for the duration of your Developer Program membership.
NVIDIA AI Enterprise: Required once you move to production. This provides enterprise-grade security, dedicated support, and higher stability for commercial applications.

import os
import requests
from openai import OpenAI

# 1. Using the OpenAI Library (Cleanest method)
client = OpenAI(
base_url="https://integrate.api.nvidia.com/v1",
api_key="nvapi-you_API-key"
)

def list_nvidia_models():
models = client.models.list()

print(f"{'Model ID':<50} | {'Owned By'}")
print("-" * 70)

for model in models.data:
# Most models in the hosted catalog are part of the 'Free' developer tier
print(f"{model.id:<50} | {model.owned_by}")

# 2. Using Requests (Best for raw data/filtering)
def get_raw_model_list():
url = "https://integrate.api.nvidia.com/v1/models"
headers = {
"Authorization": "Bearer your_nvidia_api_key_here",
"Accept": "application/json"
}

response = requests.get(url, headers=headers)
if response.status_code == 200:
data = response.json()
# You can filter the 'data' list here based on naming conventions
# e.g., models with "instruct" or specific versions
return data, data['data']
return []

list_nvidia_models()

Let’s build a quick CLI chat application, that will save the conversation history, using a reasoning model for free from NVIDIA NIM.

Packages

We don’t need a virtual environment, because these libraries are used everywhere:

pip install rich requests openai
The chat app (in the terminal)

This Python tool transforms a basic API connection into a polished, terminal-based chat application.

If you are writing about free or high-performance LLM providers, NVIDIA is a standout because of its speed and the variety of open-weight models (like Llama 3, Mistral, and DeepSeek) available to test for free through their Integrate API.

🌟 Here the features I created at a glance
Interactive Model Selection: Choose your “brain” from a list of top-tier AI models at startup.
Memory (Context Persistence): The app remembers what you said 10 minutes ago, allowing for true back-and-forth conversations.
Thinking Mode: Specialized support for “Reasoning” models (like DeepSeek or GLM) where you can see the AI’s “inner monologue” before it answers.
Smart Session Logging: Every chat is automatically saved to a unique .json file. You can stop a chat today and resume it tomorrow!
Markdown Support: Code blocks, bold text, and lists are rendered beautifully in your terminal.
Token Tracker: Real-time stats on how much “data” (tokens) your conversation is consuming.
import os
import json
import glob
from datetime import datetime
from openai import OpenAI
from rich.console import Console
from rich.markdown import Markdown
from rich.prompt import Prompt
from rich.panel import Panel
from rich.table import Table

console = Console()

MODELS = [
"stepfun-ai/step-3.5-flash",
"z-ai/glm4.7",
"moonshotai/kimi-k2-thinking",
"mistralai/mistral-nemotron",
"marin/marin-8b-instruct"
]

client = OpenAI(
base_url="https://integrate.api.nvidia.com/v1",
api_key="nvapi-yourAPIkey"
)

def get_multiline_input():
"""Allows user to paste or type multiple lines. Ends on Ctrl+D (Unix) or Ctrl+Z (Win)."""
console.print("[bold green]USER (Ctrl+D/Z to send, 'exit' to quit):[/bold green]")
lines = []
try:
while True:
line = input()
lines.append(line)
except EOFError:
pass
return "\n".join(lines).strip()

def select_session():
"""Scans for existing logs and asks to resume or start new."""
files = glob.glob("chat_log_*.json")
if not files:
return None, []

console.print(Panel("[bold yellow]Existing Sessions Found[/bold yellow]", expand=False))
console.print("0: [bold green]Start a New Session[/bold green]")
for i, f in enumerate(files, 1):
console.print(f"{i}: {f}")

choice = Prompt.ask("\nChoose a session", choices=[str(i) for i in range(len(files) + 1)], default="0")

if choice == "0":
return None, []

filename = files[int(choice) - 1]
with open(filename, "r") as f:
return filename, json.load(f)

def select_model():
console.print(Panel("[bold blue]Model Selection[/bold blue]", expand=False))
for i, m in enumerate(MODELS, 1):
console.print(f"[bold cyan]{i}[/bold cyan]: {m}")
choice = Prompt.ask("\nSelect model", choices=[str(i) for i in range(1, len(MODELS) + 1)], default="1")
return MODELS[int(choice) - 1]

def main():
# 1. Session & Model Setup
log_filename, history = select_session()
if not log_filename:
timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
log_filename = f"chat_log_{timestamp}.json"
history = [{"role": "system", "content": "You are a helpful assistant."}]

selected_model = select_model()
console.print(f"\n[bold yellow]Active Session:[/bold yellow] [green]{log_filename}[/green]\n")

try:
while True:
user_input = get_multiline_input()
if not user_input: continue
if user_input.lower() in ["exit", "quit"]: break

history.append({"role": "user", "content": user_input})

# API Call with Usage Tracking
completion = client.chat.completions.create(
model=selected_model,
messages=history,
stream=True,
stream_options={"include_usage": True}, # Crucial for token tracking
extra_body={"chat_template_kwargs": {"enable_thinking": True, "clear_thinking": False}},
)

full_content = ""
is_thinking = False
usage_data = None

for chunk in completion:
# Handle Usage Chunk (usually the last chunk)
if hasattr(chunk, 'usage') and chunk.usage is not None:
usage_data = chunk.usage
continue

if not chunk.choices: continue
delta = chunk.choices[0].delta

# Reasoning
reasoning = getattr(delta, "reasoning_content", None)
if reasoning:
if not is_thinking:
console.print("\n[bold magenta]█ THINKING[/bold magenta]")
is_thinking = True
console.print(f"[grey50]{reasoning}[/grey50]", end="", highlight=False)

# Content
if getattr(delta, "content", None) is not None:
if is_thinking:
console.print("\n\n[bold cyan]█ RESPONSE[/bold cyan]")
is_thinking = False
console.print(delta.content, end="", highlight=False)
full_content += delta.content

# Turn Summary
if full_content:
console.print("\n" + "─" * console.width)
console.print(Markdown(full_content))

# Token Usage Summary
if usage_data:
table = Table(title="Token Usage", show_header=False, box=None)
table.add_row("Prompt Tokens", f"[cyan]{usage_data.prompt_tokens}[/cyan]")
table.add_row("Completion Tokens", f"[cyan]{usage_data.completion_tokens}[/cyan]")
table.add_row("Total Tokens", f"[bold green]{usage_data.total_tokens}[/bold green]")
console.print(table)

console.print("─" * console.width + "\n")
history.append({"role": "assistant", "content": full_content})

with open(log_filename, "w") as f:
json.dump(history, f, indent=4)

except KeyboardInterrupt:
console.print("\n[bold red]Interrupted. Saving and exiting...[/bold red]")
finally:
console.print(f"[bold yellow]History saved to {log_filename}[/bold yellow]")

if __name__ == "__main__":
main()

🛠️ How It Works

You don’t need to be a python expert to understand this script. It follows a simple Loop of Life:

1. The Setup (select_session & select_model)

When you run the script, it first looks around your folder. If it sees old chat files (chat_log_...json), it asks if you want to continue an old conversation. Then, it lets you pick which AI model you want to talk to.

2. The Multiline Input (get_multiline_input)

Standard chat apps often break if you try to paste a long email or a block of code. This function allows you to paste as much as you want. You simply hit Ctrl+Z and Enter (on Windows) or Ctrl+D (on Linux/Mac) when you’re done typing to send the “envelope” to NVIDIA.

3. The Stream (completion)

Opencode is an interactive CLI tool that helps users with software engineering tasks. It can be launched via [[Bifrost-CLI]] which automatically configures it to connect through a [[Bifrost]] gateway.

## Key Features

- Interactive CLI for coding assistance
- Supports connections through [[AI Gateways]] like Bifrost
- Can be configured with zero manual setup via Bifrost-CLI

## Related Tools

- [[Bifrost-CLI]]: Can launch Opencode with automatic configuration
- [[Bifrost]]: AI gateway that Opencode can route through
- [[Claude Code]]: Alternative coding agent
- [[Coding Agents]]: Category of AI tools for software development

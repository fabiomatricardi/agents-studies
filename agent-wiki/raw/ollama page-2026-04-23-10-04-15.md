## Quickstart

Visit on a desktop computer to download Ollama and get started.

[Browse models →](/search)

### Download

Download Ollama to run models on your local machine or in the cloud.

`irm https://ollama.com/install.ps1 | iex`

Or [download manually](/download/OllamaSetup.exe)

---

### Get started

Start coding, chatting, and automating by launching the Ollama terminal UI.

`ollama`

Navigate with ↑/↓, press enter to launch, → to change model, and esc to quit.

The menu provides quick access to run a model, launch tools, and more.

or launch directly

OpenClaw Coding Chat

`ollama launch claude`

[View all coding integrations →](https://docs.ollama.com/integrations)

`ollama run gemma3`

[Browse all models →](/search)

`ollama launch openclaw`

[See documentation →](https://docs.ollama.com/integrations/openclaw)
